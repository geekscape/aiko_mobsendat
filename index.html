aiko_mobsendat by geekscape

Rocket avionics for MobSenDat (Arduino based telemetry board)

Dependencies

https://github.com/geekscape/Aiko

Install

  • 1) Download Arduino IDE
  • 2) mkdir -p ~/arduino/sketchbook/libraries
  • 3) cd ~/arduino/sketchbook/libraries
  • 4) git clone https://github.com/geekscape/Aiko.git
  • 5) cd ~/arduino/sketchbook
  • 6) git clone https://github.com/geekscape/aiko_mobsendat.git
